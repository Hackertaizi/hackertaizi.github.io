发表于 2023-09-16 更新于 2023-09-28 分类于 Java ， Java中间件 ， Maven 阅读次数： 本文字数： 615 阅读时长 ≈ 1 分钟

maven安装

  • 官网下载maven压缩包，link下的zip包
  • 解压安装包到特定目录
  • 添加系统环境变量，命名为MAVEN_HOME,路径为解压后的文件夹目录
  • 在Path中添加路径，MAVEN_HOME下的bin目录，即：

%MAVEN_HOME%\bin
