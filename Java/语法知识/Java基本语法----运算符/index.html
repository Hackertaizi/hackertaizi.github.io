一些运算符使用的注意点

在逻辑运算符中，存在短路机制，如计算 a&&b,如果a为假，则结果必为假，此时程序不会去计算b。直接给出结果，这就是短路机制

位运算由于效率很高常用于算法中，左移一位相当于乘以2，右移一位相当于除以2.
