Java语法基础—— 类型转换

Java是强类型语言，运算时，不同类型的值要先转化为同一类型。

数据类型容量

强制转换

语法： (类型)变量

使用情景： 高容量类型变量—>低容量类型变量

自动转换

低容量类型变量—>高容量类型变量时会自动执行数据类型转换

注意

  • 不能对布尔类型进行转换
  • 不能将对象类型转换为不相干的类型
  • 注意在类型转换时的内存溢出和精度问题

int类型特性

整数数值可以使用下划线做分隔符，不会改变数值

如1000000=100_0000
