异常处理机制

捕获异常

使用try和catch关键字可以捕获异常，try/catch代码块放在可能发生异常的地方，其语法格式如下：

try后面可以跟多个catch块，哪个catch块声明的异常发生，哪个catch块就捕获到异常，值得一提的是，catch块的顺序不是随意的，要根据异常类的层次从小到大书写，否则后面的catch块会被前面的catch块所覆盖，导致错误。

throw/throws关键字

throw关键字用于主动抛出异常，语法格式为：throw new 异常类（）；

如果一个方法没有捕获到一个检查性异常，那么该方法必须使用 throws 关键字来声明。throws 关键字放在方法签名的尾部。

ctrl+alt+T

选中代码块，使用上述快捷键，可以快速将代码块使用关键字包裹（例如 if，else，try，catch等）

finally关键字

finally代码块用来放在try/catch代码块的最后面，无论是否发生异常，finally代码块中的代码总会被执行，finally代码块中主要写一些处理善后性质的语句。

自定义异常

java中可以自定义异常，需要记住一下几点：

  • 所有异常都必须是Throwable的子类
  • 如果要自定义检查性异常类，需要继承Exception类
  • 如果要自定义运行异常类，需要进程RuntimeException类
