面向对象

类和对象

类是对象的抽象，类是对象的模板，对象是类的示例，类实例化后就是对象。

创建和初始化对象

创建对象使用new关键字，语法为：类名 对象名 = new 类名();

使用new关键字创建的时候，处理分配内存空间之外，还会给创建好的对象进行默认的初始化以及对类中构造器的调用。

构造器

特点

类中的构造器也称为构造方法，是在进行创建对象的时候必须要调用的。构造器有如下两个特点：

  • 1 . 必须和类的名字相同
  • 2 . 没有返回类型，也不写void

作用

  • 使用new关键字，本质上是在调用构造器。
  • 可以在构造器中初始化对象的值。

注意

一旦定义了有参数的构造器，如果要使用无参数的构造器，就必须显示定义无参数的构造器。有参数的构造器是对无参构造器的重载。

快捷键

Alt+insert快速创建构造器。

面向对象的特点

封装

使用private标识符可以将类中的属性和方法设置为私有类，从而不能被直接调用，可以定义某些方法来作为调用私有属性和类的接口。

特性

  • Java中只有单继承，没有多继承。
  • 继承是类和类之间的一种关系。除此之外，类和类之间的关系还有依赖、组合、聚合等。
  • 子类继承父类，也就继承了父类中的所有公有属性和方法。

super方法

  • super调用父类的构造方法，必须在子类构造方法的第一行
  • super必须只能出现在子类的方法或构造方法中
  • super和this不能同时调用构造方法，因为this也必须在构造方法的第一行

特殊

object类默认是所有类的父类

方法重写

重写的意义

当某些父类方法的功能不能满足子类的需要时，子类需要对父类的方法进行重写

重写规则

  • 重写是建立在有几成关系的两个类间的，子类可以重写父类的方法。

  • 要重写的方法是非静态方法，静态方法不会被重写成功

  • 要重写的方法的方法名和参数列表必须和父类的相同

  • 修饰符：范围可以扩大不可以缩小(public>protected>default>private)

  • 抛出的异常：范围可以缩小不能扩大

多态

概念

同一方法可以根据发送对象的不同而采用多种不同的行为方式。多态是方法的特性，属性没有多态性。
