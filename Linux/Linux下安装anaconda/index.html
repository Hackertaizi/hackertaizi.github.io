Linux下安装anaconda

  • 首先在anaconda官网下载Linux版本的anaconda的shell包；
  • 使用bash命令执行.sh脚本文件安装anaconda，注意更改安装路径，一般可以安装在/opt目录下；
  • 安装结束后，在系统中添加环境变量：

sudo vim /etc/profile

在配置文件最后加入路径：

#Anaconda

export PATH=$PATH:anaconda安装路径/bin
