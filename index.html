1.梯度下降

梯度

在单变量函数中，梯度是函数的微分，代表函数在某点切线的斜率；在多变量函数中，梯度是每个变量的微分组成的向量，向量的方向是函数增长最快的方向，其反方向是函数下降最快的方向。

2.批量梯度下降法(Batch Gradient Descent，BGD)

算法思想

批量梯度下降的思想是每次迭代时使用全部样本进行梯度更新，具体说来，对于目标函数和给定的目标点，首先我们计算该点的梯度，然后依照梯度公式对该点进行更新，注意更新时使用的是所有样本的平均梯度。

优缺点

优点：

1.一次迭代过程中是对所有样本进行了计算，此时利用矩阵进行操作，实现了并行。

2.由全数据集确定的方向能够更好地代表样本总体，从而更准确地朝极值所在的方向。当目标函数为凸函数时，该算法一定能够得到全局最优点。

缺点：

1.当样本数目很大时，由于每次迭代需要所有样本都参与计算，所以训练过程会很慢。

3.随机梯度下降法(Stochastic Gradient Descent，SGD)

随机梯度下降每次进行梯度迭代时只是用一个样本对参数更新，随机优化一条训练数据上的损失函数。

优缺点

优点：

由于每次迭代只是用一个样本，相比于批量梯度下降法，训练速度要快很多。

缺点：

1.准确度下降，即使目标函数为凸函数，该算法仍然无法做到线性收敛。

2.可能会收敛到局部最优值，因为单个样本不能代表全体样本的趋势。

3.不易于并行实现。

4.小批量梯度下降(Mini-Batch Gradient Descent, MBGD)

小批量梯度下降是对批量梯度下降和随机梯度下降的一种折中的方法，它每次迭代使用batch_size个样本对参数进行更新。

优缺点

优点：

1.通过矩阵运算，每次在一个batch上优化神经网络参数并不会比单个数据慢太多。

2.每次使用一个batch迭代可以减少训练所需的迭代次数，同时可以使收敛的结果更加接近于梯度下降的效果。

3.可以实现并行。

缺点：

1.batch_size的大小选取不当可能会带来问题。

深度学习基础

一、基础知识

包括线性代数、概率论、信息论、数值计算等知识，此处不再赘述。

二、关于机器学习

1.容量

模型的容量是指模型的拟合能力。

量化模型容量的方法：VC维。

2.过拟合和欠拟合

过拟合是指模型在训练集上学习到了太多细微的特征，导致模型在其他数据集上的预测表现很差。即训练误差很小，但泛化误差（测试误差）很大。

欠拟合是指模型在训练集上的表现很差，集训练误差较大。

3.模型训练与优化的区别

优化的目标是降低模型的训练误差，而训练的目的不但要降低训练误差，更主要的是降低泛化误差，即模型在测试集上的误差（在没有使用过的数据集上的预测误差）。

4.只在训练集上训练的模型，在测试集上的泛化误差如何被影响？

在收集数据时，我们一般假设训练集和测试集是独立同分布的，这样，当我们固定参数时，模型在两个数据集上的表现是相同的（当然实际模型训练时不会固定两个数据集的参数相同）。

5.超参数

定义：人为预先设定的、能够控制算法行为的参数，无法同过算法学习获得。

6.训练集、测试集、验证集

训练集用于训练模型；测试集从训练集中划分，用于衡量模型的预测能力，所以测试集不参与模型的构建过程，包括超参数的设置；在训练集上学习超参数时，其总是趋向于最大的模型容量，从而导致过拟合，因此引入验证集解决此问题，鉴于测试集的性质，验证集只能从训练集中划分。验证集用于估计训练过程中和训练后的泛化误差，来更新超参数，即验证集主要是更新超参数的，虽然也能估计泛化误差，但其常常会低估泛化误差，所以会通过测试集来进一步估计泛化误差。

7.交叉验证

在模型训练中，训练数据集规模越大，训练的效果越好。一般训练时将数据集划分成了多个部分，只有一部分用于训练，这样得到的模型无法发挥数据集的规模优势；而且数据集的划分方法也会对模型的训练和预测误差产生影响（不同的划分比例，训练集与原模型是否有相同的比例分布等都会影响模型的效果）。为了解决这些问题，提出了交叉验证法。

K-折交叉验证：

将数据集不重复抽样随机分为K份，其中一份作为测试集，其余部分作为训练集进行训练，将上述部分重复执行K次，直至每个部分都做过测试集。最后将K组测试的结果做平均，作为交叉验证的最终结果。

留一法：

即K折交叉验证中K等于数据集规模的情况，每一份划分只有一个数据，每次选取一个数据用于测试，其余数据用于训练，这样能够得到最贴近原始数据集的训练结果，但是训练复杂度较大。留一法适用于数据集规模较小的情况。

8.随机梯度下降法

梯度下降

导数公式f(x + ϵ) ≈ f(x) + ϵf′(x)，表明了如何缩放输入的小变化才能在输出得到相应的变化，当我们需要求目标函数的最小值（即代价函数）时，我们可以通过将x向目标函数的导数反方向移动来减小目标函数，这种技术叫做梯度下降。

反向传播算法

前向传播和反向传播的概念

在神经网络中，输入信息x通过输入层、中间层、一直向前传递直到产生输出，这个过程叫做前向传播。在训练过程中，信息一直向前传播直到产生一个代价函数。反向传播的思想是让来自代价函数的信息通过网络向后流动，以便计算梯度。

反向传播

1.2 二维卷积层

卷积神经网络常用来处理图像数据，二维卷积层将输入的二维图像和卷积核做互相关运算，然后加上一个偏差来得到输出。卷积核和偏差是超参数。在训练模型时，通常先对卷积核做随机初始化，然后不断迭代卷积核和偏差。

1.3 特征图和感受野

特征图是输入在空间维度上某一级的表征，也即二维卷积层的输出。感受野是特征图中某一点在输入矩阵对应的一块区域。

1.4 填充和步幅

对于一个卷积层来说，设输入形状是x1×x2,卷积核形状是k1×K2，则输出特征图的形状将是：

                                                             (x1-k1+1)×(x2-k2+1)

即卷积层的输出形状由输入形状和卷积核形状决定，除此之外，卷积层的两个超参数填充和步幅可以在输入形状和卷积核形状确定的情况下改变输出的形状。

填充是指在输入的高和宽两侧填充元素，一般填充的是0元素。一般来说，如果在高的两侧填充h行，在宽的两侧填充w列，输出形状将是：

​ (x1-k1+h+1)×(x2-k2+w+1)

步幅即卷积核在数据矩阵上每次滑动的距离，当高上的步幅为sh，宽上的步幅为sw时，输出形状为：

​ ⌊(x1-k1+h+1)⌋/sh×⌊(x2-k2+w+1)⌋/sw

​

1.5 多输入/输出通道

输入数据有多个通道时，需要构造通道数相同的卷积核。最后按通道相加得到输出，示例图如下：

输入通道有多个时，由于我们对每个通道的输出进行了累加，所以输出通道总是1。如果希望得到多通道的输出，我们设输入通道数为in，输出通道数为out，为了匹配输入通道数，核数组形状应该为k1×k2×in,为了继续匹配输出通道数，卷积核的最终形状应该为k1×k2×in×out,即对于每个输出通道，我们都为其分配一个形状为k1×k2×in的卷积核。

1.6 1×1卷积层

当卷积核的形状为1×1时，相应的卷积层称为1×1卷积层，在宽和高两个维度上 ，1×1卷积层无法识别相邻元素的构成模式，1×1卷积的主要计算发生在通道维上，输出的形状与输入的形状相同，输出中的每个元素都是输入中每个元素在不同通道上按权重的累加。假设我们将通道维当作特征维，将高和宽维度上的元素当成数据样本，那么1×1卷积层的作用与全连接层等价。1×1卷积层通常用来调整网络层之间的通道数，并控制模型复杂度。

2.1 池化层

池化层的提出是为了缓解卷积层对目标在图像中的位置的过度敏感性。

池化层的操作类似于卷积层，这里的核数组称为池化窗口，不同于卷积核，池化窗口只是一个尺寸，里面没有数据，池化窗口在输入的特征图上从左上到右下滑动，一般取其中的最大值或平均值，即最大池化和平均池化。

在处理多输入通道数据时，池化层对每个通道的输入分别进行池化操作，这使得池化层的输出通道数与输入通道数相等。

二、卷积神经网络LeNet

LeNet是早期用来实现手写数字识别的卷积神经网络，网络的训练使用了梯度下降法，LeNet的网络结构如下图所示：

LeNet分为卷积层块和全连接层块两个部分，卷积层块中的基本单位是卷积层后接一个最大池化层：卷积层用来识别图像中的空间模式，池化层用来降低卷积层对位置的敏感性。在卷积层块中，每个卷积层都使用5×5的卷积核，并在输出上使用sigmoid激活函数。第一个卷积层为6通道输出，第二个卷积层为16通道输出，因为第二个卷积层的输入比第一个卷积层的输入小，所以增加第二个卷积层的输出通道数是两个卷积层的参数尺寸类似。卷积层块中两个最大池化窗口的形状都为2×2，步幅为2，池化窗口每次滑动时覆盖的区域互不重叠。

卷积层块的输出形状为：批量大小×通道数×高×宽。当卷积层块的输出传入全连接层块时，全连接层块会将小批量中每个样本变平（flatten）。也就是说，全连接层的输入形状将变成二维，其中第一维是小批量中的样本，第二维是每个样本变平后的向量表示，且向量长度为通道、高和宽的乘积。全连接层块含3个全连接层。它们的输出个数分别是120、84和10，其中10为输出的类别个数。

三、深度卷积神经网络AlexNet

AlexNet与LeNet的设计理念非常相似，但也有显著的区别。

第一，与相对较小的LeNet相比，AlexNet包含8层变换，其中有5层卷积和2层全连接隐藏层，以及1个全连接输出层。下面我们来详细描述这些层的设计。

首先给出AlexNet的网络结构：

AlexNet第一层中的卷积窗口形状是11×11。因为ImageNet中绝大多数图像的高和宽均比MNIST图像的高和宽大10倍以上，ImageNet图像的物体占用更多的像素，所以需要更大的卷积窗口来捕获物体。第二层中的卷积窗口形状减小到5×5，之后全采用3×3。此外，第一、第二和第五个卷积层之后都使用了窗口形状为3×3、步幅为2的最大池化层。而且，AlexNet使用的卷积通道数也大于LeNet中的卷积通道数数十倍。

紧接着最后一个卷积层的是两个输出个数为4096的全连接层。这两个巨大的全连接层带来将近1GB的模型参数。由于早期显存的限制，最早的AlexNet使用双数据流的设计使一个GPU只需要处理一半模型。幸运的是，显存在过去几年得到了长足的发展，因此通常我们不再需要这样的特别设计了。

第二，AlexNet将sigmoid激活函数改成了更加简单的ReLU激活函数。一方面，ReLU激活函数的计算更简单，例如它并没有sigmoid激活函数中的求幂运算。另一方面，ReLU激活函数在不同的参数初始化方法下使模型更容易训练。这是由于当sigmoid激活函数输出极接近0或1时，这些区域的梯度几乎为0，从而造成反向传播无法继续更新部分模型参数；而ReLU激活函数在正区间的梯度恒为1。因此，若模型参数初始化不当，sigmoid函数可能在正区间得到几乎为0的梯度，从而令模型无法得到有效训练。

第三，AlexNet通过丢弃法来控制全连接层的模型复杂度。而LeNet并没有使用丢弃法。

第四，AlexNet引入了大量的图像增广，如翻转、裁剪和颜色变化，从而进一步扩大数据集来缓解过拟合。

Pipeline

一、概念

在模型的构建过程中，一般存在着数据读取、数据处理，模型构建、模型评估、模型调参等步骤，这些步骤可以抽象为模型，执行从输入到输出的整个过程，这个过程称为pipeline。

二、Sklearn中的pipeline机制

sklearn中把机器学习处理过程抽象为estimator，其中estimator都有fit方法，表示数据进行初始化or训练。estimator有2种：

1、特征变换（transformer）
可以理解为特征工程，即：特征标准化、特征正则化、特征离散化、特征平滑、onehot编码等。该类型统一由一个transform方法，用于fit数据之后，输入新的数据，进行特征变换。

2、预测器（predictor）
即各种模型，所有模型fit进行训练之后，都要经过测试集进行predict所有，有一个predict的公共方法。

上面的抽象的好处即可实现机器学习的pipeline，显然特征变换是可能并行的，通过FeatureUnion实现。特征变换在训练集、测试集之间都需要统一，所以pipeline可以达到模块化的目的。

维数灾难

在高维空间中，机器学习方法会面临数据样本稀疏、距离计算困难等问题，这种问题被称为维数灾难。

低维嵌入

解决位数灾难的方法是降维，或称为维数约减，即通过某种数学变换将原始高维属性空间转变为一个低维的子空间，在子空间中样本密度大幅提高，距离计算也变得更为容易。能进行降维操作是因为在很多时候，人们观测或收集到的数据样本虽然是高维的，但与学习任务密切相关的也许仅是某个低维分布，即高维空间中的一个低维嵌入。

聚类

一、基本知识

1.什么是聚类任务？

聚类任务是针对于无监督学习任务的，用于聚类的样本没有标记信息，需要通过学习来揭示数据的内在性质和规律。

2.聚类原理

聚类一般是把数据集中的样本划分为若干个不相交的子集，每个子集称为一个‘簇’，通过划分，每个簇可能对应一些潜在的概念或类别。但是，聚类只能划分出一个个簇，每个簇对应的概念语义还需要使用者来把握和命名。

3.聚类的用途

聚类可以作为单独的过程，用于探索数据的内在分布结构，也可以作为分类等其他任务的前驱过程。

4.聚类的评价指标——性能度量

聚类的目标是让簇内样本的相似度高而簇间样本的相似度低。聚类的性能度量有两类：一是将聚类结果与某个“参考模型”进行比较，称为“外部指标”；二是直接考察聚类结果而不利用任何参考模型，称为“内部指标”。

5.聚类的依据——距离度量

在样本集中，聚类是根据距离衡量来进行的，对于不同形式的样本，有不同的距离度量方法。首先，对于一个距离度量dist()，需满足一些基本性质，即

  • 非负性：dist(x1,x2)>=0
  • 同一性：dist(x1,x2)=0当且仅当x1=x2
  • 对称性：dist(x1,x2)=dist(x2,x1)
  • 直递性：dist(xi,xj)<=dist(xi,xk)+dist(xk,xj)

5.3 有序属性与无序属性

属性常划分为连续属性和离散属性，在距离计算中，更重要的是属性是否定义了顺序。例如定义域为{1，2，3}的离散属性与连续属性的性质更接近一些，能直接在属性值上计算距离：1与2比较近，与3比较远，这样的属性称为有序属性；而定义域为{火车，飞机，轮船}这样的离散属性不能直接在属性值上计算距离，称为无序属性。显然，闵可夫斯基距离可用于有序属性。

二、聚类算法

1. 原型聚类

“原型”指的是样本空间中具有代表性的点。原型聚类假设聚类结构能够通过一组原型刻画。通常情况下，原型聚类算法先对原型进行初始化，然后对原型进行迭代更新求解。采用不同的原型表示、不同的求解方式，将产生不同的算法，以下是几种著名的原型聚类算法。

1.1 k均值算法（k-means算法）

k-means算法的目标是将无标签样本最终划分为k个簇，即分成k类。算法首先先随机选取k个原型样本，然后按照距离度量将距原型样本最近的点集合划分成一个簇。最终样本集会根据选取的k个原型点聚类成k个簇。簇划分完毕后，计算簇内样本的均值，即簇的中心点，如果中心点与原型样本点不同，则将原型点更改为中心点。然后以更新的原型点为聚类点重复上述过程，直到所有的原型点都不再更新，聚类结束。

1.2 学习向量量化（Learning Vector Quantization,LVQ)

LVQ算法也试图找到一组原型向量来刻画聚类结构，不同于k-means的是，LVQ假设数据样本带有类别标记，学习过程利用样本的这些监督信息来辅助聚类。

LVQ算法首先从每个类别标记中选取一个样本作为原型向量，然后对其余样本按照距离度量进行聚类，如果聚类的样本与原型点的类别标记相同，则令原型向量向该样本靠拢，否则远离该样本（即原型向量的更新）。重复该过程，直到满足停止条件（已达到最大迭代轮数，或者原型向量更新很小甚至不再更新）。最终得到一组原型向量。

最(极)大似然法

1.用极大似然法进行参数估计/最大似然估计(Maximum likelihood estimation, MLE)

极大似然法的估计场景是模型已知而参数未知的情况，也就是使用已知的样本结果信息来反推出最有可能(概率最大)导致这些样本结果出现的模型参数值。

使用极大似然估计的前提是所有样本的采样必须都是独立同分布的。

关于概率和统计

概率问题是已知模型和参数，求结果数据。

统计问题时已知数据，求模型和参数。

极大似然

首先给出一个函数形式P(x|θ)

这是一个条件概率函数，如果θ是已知的，x是未知的。则该函数称为概率函数。表示对于不同的样本点x，其出现概率的大小。

如果x是已知的，θ是未知的，该函数称为似然函数。表示对于不同的参数θ，出现样本点x概率的大小。

对于极大似然，使用一个例子来解释。设一个罐子中有若干白球和黑球，现要求白球与黑球的比例，每次取出若干球并记录其颜色，然后放回罐子。重复这个过程，通过记录的颜色数据来推断黑白球的比例。假设每次取一个球，取100次，其中白球出现了70次，那么我们可以推测白球占70%，其理论依据是我们抽取的数据样本是独立同分布的。设白球出现的概率为p，第n次抽取结果为xn，则抽取100次出现70次白球的概率是：P(样本结果|模型参数)=P(x1,x2…x100|模型参数)=P(x1|模型参数)* P(x2|模型参数)*…P(x100|模型参数) = p·70·(1-p)·30

现在我们已经拥有了结果的概率表达式了，如何求式中的p呢?p具有无数种分布，极大似然法的思想是令当前结果的分布概率出现最大，即让p·70·(1-p)·30的出现概率最大。

这时只需要对概率公式求导，并令导数为0，即可求得p=70%

总结：

对于要估计的目标，我们先假定其具有某种概率分布P(x|θ)，然后再基于训练集D对概率分布的参数θ做估计。我们假设这些样本是独立同分布的，参数θ对于数据集D的似然是：P(D|θ)=∏P(x|θ)（x∈D）
